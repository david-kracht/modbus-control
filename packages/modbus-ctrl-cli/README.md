# Modbus Control CLI

The `modbus-ctrl-cli` package provides a robust command-line interface and interactive TUI (Text User Interface) for configuring and polling Modbus devices.

## Purpose

- Parses command-line arguments and handles environment variables.
- Renders an interactive, visually rich Terminal UI using `rich`.
- Integrates seamlessly with `modbus-ctrl-core` to present real-time polling data and facilitate register manipulation.
